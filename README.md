# TaskManager

## Описание
Приложение для управления задачами (WinForms).

## Возможности
- Добавление задач
- Редактирование задач
- Удаление задач
- Сортировка задач
- Фильтрация задач
- Цветовая индикация просроченных задач

## Требования:

- Windows
- .NET 6/8
- Visual Studio 2022+

## Запуск
1. Открыть файл решения (.sln или .slnx)
2. Нажать F5
